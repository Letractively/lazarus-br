Cheques 2.1 - Controle pessoal de cheques
=========================================

Projeto simples para estudos sobre Object Pascal e orientação a objetos.

O projeto contém scritp para criação de setup para Windows, bem como distribuição
de arquivos para criação de arquivos DEB.

Contribuíram diretamente neste projeto Luiz Americo e Silvio Clécio, e indiretamente
Joao Morais.

O autor inicial do projeto contribuiu com os requisitos.
