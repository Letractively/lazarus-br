Neste exemplo mostro três formas de exibir formulários de bibliotecas:

Exibir_Form_de_Biblioteca: Este exemplo simplesmente exibe o formulário.

Exibir_Form_de_Biblioteca_em_TPanel: Este exemplo exibe o formulário de um uma biblioteca dentro de outro formulário na aplicação. Este exemplo não funciona bem no Windows.

Exibir_TFrame_de_biblioteca: Este exemplo faz o mesmo que o exemplo anterior, mas sem erros, por usar TFrame no lugar do TForm.
